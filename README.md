# IoT Predictive Maintenance Node using Multicore Edge AI
The IoT Predictive Maintenance Node is an advanced industrial monitoring system designed to ensure the safety and operational efficiency of industrial devices through real-time fault detection. By leveraging an ESP32 microcontroller and Edge AI, the system implements a localized machine learning approach that eliminates the high latency and bandwidth dependencies typically found in cloud-based monitoring solutions. The architecture utilizes the dual-core 32-bit LX6 microprocessor to perform asymmetric multiprocessing via FreeRTOS, effectively separating high-priority deterministic tasks from asynchronous networking routines. Core 1 is dedicated to high-frequency data acquisition from an MPU6050 vibration sensor at a sampling frequency of 100 Hz, while Core 0 manages the Wi-Fi stack and cloud telemetry to ThingSpeak. This separation is maintained through thread-safe FreeRTOS Queues, which prevent race conditions and ensure that the neural network does not lose mechanical frames during network activity.

The system processes 3-axis kinematic data through a sophisticated TinyML pipeline that includes non-linear software filtering to mitigate baseline drift and high-frequency mechanical spikes. Digital Signal Processing (DSP) via Fast Fourier Transform (FFT) is applied to the input tensor, allowing the Sequential Keras Classifier to identify separable spectral peaks associated with shaft misalignment and mounting faults. To fit within the ESP32’s hardware constraints, the model underwent aggressive 8-bit integer quantization, which reduced the flash memory footprint to 16.0 KB and accelerated inference latency to just 1 ms. Operational dependability is further reinforced by a multi-layered heuristic decision matrix, which includes a physical amplitude gate to bypass erratic predictions at low signal levels and a temporal majority voting algorithm. A 3-strike fault tolerance logic ensures that the system only triggers a 5V relay to shut down the motor when a critical failure is confirmed across multiple independent classification windows.

Experimental results demonstrate that the trained model achieves a high classification accuracy of 98.9% with a validation loss of 0.06. Feature visualization confirms that vibration signatures for idle, normal, and faulty motor conditions are well-distinguished, enabling the system to correctly identify primary faults even in overlapping conditions. By transitioning from a basic breadboard implementation to a fully integrated industrial-grade module, this project provides a cost-effective and scalable predictive maintenance solution suitable for Industry 4.0 applications. The final implementation ensures stable operation without internet connectivity while providing off-site supervisors with real-time asset degradation data via web-based dashboards.
